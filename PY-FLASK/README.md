### Install Flask
```
pip install flask
```

### Run Flask App
```
# Set Script to run
export FLASK_APP=main.py

# Run Flask app
flask run
```

### Start app in Debug Mode
```
# Set Script to run
export FLASK_APP=main.py

# Set Degug Mode
export FLASK_DEBUG=1

# Run Flask app
flask run
```

### For JWT token in API, install PyJWT
```
pip install pyjwt
```
